# OrbitPulse アイディア帳

本プロジェクト「OrbitPulse」をより良く、楽しく、そして自分らしくするためのアイディア集。

## 1. ゲーミフィケーション & モチベーション
- **Pulse（パルス）システム**:
    - ポモドーロ（25分）を「1 Pulse」という単位で呼ぶ。
    - 数字に縛られすぎず、その日のリズム（鼓動）に合わせて1パルスの長さを微調整できても良い。
    - 集中が続いている時は「ボーナタイム（オーバーフロー）」として継続を許容する。
- **成長要素**:
    - 完了したストーリーポイントを経験値（XP）に見立て、ユーザーレベルが上がる仕組み。
    - 週末に「今週のボス（積み残しタスク）」を倒す演出。

## 2. 人間らしさの許容（ゆるさ）
- **コンディション連動キャパシティ**:
    - 毎朝の気分や体調（絶好調、普通、低空飛行など）を入力。
    - その日の体調に合わせて、AIがスプリントバックログの負荷を自動調整・提案してくれる。
- **休憩の質を記録**:
    - 単なる「休み」ではなく、何をしてリフレッシュしたか（散歩、コーヒー等）を記録し、後の集中力との相関を分析する。

## 3. AIスクラムマスター (MCP活用)
- **対話型イベント**:
    - **プランニング**: 過去のベロシティと「Why」を考慮したストーリー提案。
    - **デイリースクラム**: 前日の振り返りと今日の予定をチャット形式で報告・相談。
    - **レトロスペクティブ**: 1週間のデータの可視化と、AIによる改善アドバイス。
- **伴走者としてのAI**:
    - 罪悪感なくタスクを次スプリントに回せるよう相談に乗る。
    - 「今は休むのも仕事」と適切なタイミングで休憩を促す。

## 4. UI/UX のこだわり
- **「Why」の強調**:
    - タスク一覧や詳細で、「なぜこれを作るのか」が常に目に入るデザイン。
- **直感的な見積もり**:
    - 数字の入力ではなく、スライダーや「天体の大きさ」などで感覚的にポイントを振る。
- **Optimistic Updates**:
    - 操作した瞬間に画面が反応する「軽快さ」の徹底。
