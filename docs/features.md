# 主要機能

OrbitPulse は「個人のためのスクラム管理」を支援するツールです。

## 1. スプリント管理
- **計画 (Planning)**: 期間を設定し、バックログからアイテムを選定します。
- **実行 (Execution)**: スプリントを開始し、日々のタスクを消化します。
- **振り返り (Retrospective)**: 終了後に気づきを記録し、次のスプリントへ繋げます。

## 2. バックログ管理
- **Why (なぜ)** にフォーカスしたストーリー作成。
- ストーリーポイントによる見積もり。
- 優先度付けとステータス管理（Backlog / Todo / Doing / Done）。

## 3. ポモドーロタイマー（Pulse）
- 25分の集中時間と5分の休憩時間を1パルスとして管理。
- タスクに紐付けて実行することで、正確な実績時間（パルス数）を記録可能。
- 集中終了時にデスクトップ通知（ブラウザ機能）を送ります。

## 4. バーンダウンチャート
- 残りパルス数の推移をグラフ化。
- スプリント期間内での進捗を視覚的に把握。

## 5. キャパシティ管理
- 日ごとの作業可能量を設定。
- 計画時点でキャパシティオーバーしていないか警告を表示。

## 6. 多言語対応 (i18n)
- 日本語と英語に対応。
- URL パスによる言語切り替え（`/ja/...`, `/en/...`）。
- メッセージは JSON ファイルで管理。
